Useful shell commands
---------------------

Find all expressions:
rg -o '[{][{][^{]*[}]' mytheme/ | cut -d: -f2 | sort | uniq

Find all commands:
rg -o '[{][%][^{]*[}]' mytheme/ | cut -d: -f2 | sort | uniq

Find all variables:
rg -o '[{][{%][^{]*[}]' mytheme/ | cut -d: -f2 | sort | uniq | grep -v include | grep -v extends | grep -v block | grep -v endfor | grep -v endif | grep -v else
