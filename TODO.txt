main logic:
-----------
articles := sort articles in site/blog/* by date
load article template
for each article, in date desc order:
    render article content from MD to HTML
        https://pkg.odin-lang.org/vendor/commonmark/#markdown_to_html
    render into HTML content using the HTML template
    update feed files (better than requiring memory everytime)
    save to output file
        create dir output tree as needed
generate pages w/ similar logic, except no feed
generate index page
    generate summary for every article
